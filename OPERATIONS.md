# 📖 운영자 가이드 (비개발자용)

> 이 문서는 **코딩을 모르는 운영자**를 위한 안내서입니다.
> 챗봇에 자료를 추가하고, 잘 작동하는지 확인하고, 문제가 생겼을 때 대처하는 방법을 다룹니다.

---

## 🧭 이 시스템이 뭔가요?

수강생이 **카카오톡 채널**(라스트원 클래스도우미)에 질문하면, 챗봇이 **등록된 자료 안에서** 관련 내용을 찾아 답변합니다.

```
수강생 질문 → 챗봇이 자료에서 검색 → 자료 내용으로 답변 (+출처 표시)
```

중요한 특징:
- 챗봇은 **올려둔 자료에 있는 내용만** 답합니다. 자료에 없으면 "자료에 포함되어 있지 않습니다"라고 솔직하게 말합니다.
- 그래서 **좋은 자료를 올리는 것 = 챗봇이 똑똑해지는 것**입니다.
- 자료를 올리면 **즉시 반영**됩니다. 별도 작업 필요 없습니다.

---

## 🔑 관리자 페이지 접속

1. 브라우저에서 **kakao-bot-admin.vercel.app** 접속
2. 이메일·비밀번호로 로그인
3. 비밀번호를 잊었다면 → 개발 담당자에게 문의

> 비밀번호는 좌측 메뉴 **설정**에서 직접 변경할 수 있습니다.

---

## 📁 자료 올리기 (가장 자주 하는 일)

### 올릴 수 있는 파일

| 파일 종류 | 되나요? | 메모 |
|---|---|---|
| PDF | ✅ | 글자가 들어있는 PDF만. 스캔(사진) PDF는 안 됨 |
| PPT (.pptx) | ✅ | 글자 위주 슬라이드만. 이미지 위주는 안 됨 |
| 한글 (.hwp) | ✅ | 최신 형식만 |
| 텍스트 (.txt) | ✅ | 자유 형식. 질문/답 형식이면 더 좋음 (아래 참고) |
| **자막 (.vtt)** | ✅ **추천** | Zoom 녹화하면 자동 생성됨. 긴 강의는 이걸로! |
| 음성 (.mp3) | ⚠️ | 짧은 것만 (몇 분 이내). 50MB까지 |
| 영상 (.mp4) | ❌ 받지 않음 | 전사가 너무 오래 걸려 제외. **자막(VTT)으로 올리세요 → 아래 "강의 영상 올리기"** |

### 올리는 방법

1. 좌측 메뉴 **자료 관리**
2. 점선 박스에 파일을 **끌어다 놓기** (여러 개 한 번에 가능)
3. 목록에서 파일 확인 → **[N개 업로드]** 버튼 클릭
4. 진행 막대가 차오르며 처리됨 (보통 몇 초~1분)
5. "✅ N개 청크 생성" 뜨면 완료 — **그 순간부터 챗봇이 이 자료로 답변**

> "청크"란? 자료를 검색하기 좋게 작은 조각으로 나눈 것입니다. 청크 수가 많을수록 내용이 많다는 뜻일 뿐, 신경 쓸 필요 없습니다.

### 🎬 강의 영상 올리기 (중요!)

영상 파일을 직접 올리면 **안 됩니다** (용량 제한). 대신 **자막 파일**을 올리세요:

**Zoom으로 녹화한 강의:**
1. Zoom 웹사이트 → 기록(Recordings) → 해당 녹화
2. **"오디오 텍스트 변환(Audio Transcript)"** 파일(.vtt) 다운로드
3. 그 .vtt 파일을 관리자 페이지에 업로드

→ 1시간 강의도 자막 파일은 아주 작아서 바로 처리됩니다.

**YouTube에 올린 강의:** YouTube 스튜디오에서 자막 다운로드 → 업로드.

### 📝 자주 묻는 질문(FAQ) 만들기 — 효과 최고

수강생이 자주 묻는 질문은 **텍스트 파일로 직접 만들어 올리는 게** 가장 정확합니다.

메모장을 열고 이런 형식으로 작성 (질문 사이에 **빈 줄 한 줄** 필수):

```
Q: 사업자등록증 재발급은 어떻게 받나요?
A: 홈택스에서 민원증명 메뉴 → 사업자등록증명원 발급. 즉시 무료 발급됩니다.

Q: 수업 결석하면 보강 가능한가요?
A: 녹화본이 노션에 올라가니 노션에서 다시 보실 수 있습니다.
```

`FAQ-사업자등록.txt` 같은 이름으로 저장 → 업로드. 끝.

### 자료 수정/삭제

- **삭제**: 파일 오른쪽 점 3개(⋮) → 삭제. 챗봇이 더 이상 그 자료로 답하지 않음
- **갱신**: 옛 파일 삭제 → 새 파일 업로드
- **카테고리 수정**: 카테고리 배지 클릭 → 올바른 분류 선택 (자동 분류가 가끔 틀림)

## 🧪 챗봇이 잘 답하는지 확인하기

좌측 메뉴 **답변 테스트**:

1. 질문 입력 (수강생이 물을 법한 질문)
2. 답변 + **"참고한 자료"** 가 같이 표시됨
3. 엉뚱한 자료를 참고했다면 → 그 주제의 자료가 부족하다는 신호 → FAQ 추가

> 여기서 한 테스트는 실제 수강생에게 보이지 않습니다. 마음껏 테스트하세요.

---

## 📊 통계 보기

좌측 메뉴 **통계**:
- 수강생들이 **얼마나, 무엇을** 묻는지
- **"답변 못한 질문"** ← 여기가 핵심! 이 질문들에 대한 자료를 만들어 올리면 챗봇이 점점 똑똑해집니다

## 💬 사용자 피드백 (할 일 목록처럼 쓰세요)

좌측 메뉴 **사용자 피드백**:
- **자료 없음**: 수강생이 물었는데 자료가 없어 답 못한 질문
- **오류**: 시스템 문제로 답변 실패한 기록 (개발 담당자에게 전달)

처리 방법:
- 자료 추가해서 해결했으면 → **[해결]** 버튼
- 여러 개 한 번에 → 체크박스 선택 → **[선택 해결]**
- 의미 없는 항목(테스트, 장난 질문) → **삭제**

## 🚨 문제가 생겼을 때

### 챗봇이 "일시적으로 답변을 생성하지 못했습니다"만 반복

시스템 오류입니다. 운영자가 할 수 있는 확인:
1. 관리자 페이지 → **사용자 피드백** → "오류" 항목이 쌓여있는지 확인
2. 오류 내용을 캡처해서 **개발 담당자에게 전달**

(주요 원인: Gemini API 키 만료 — 개발자가 키를 교체해야 합니다)

### 챗봇 답이 이상하거나 옛날 정보를 말함

- 옛날 자료가 남아있을 가능성 → **자료 관리**에서 해당 파일 찾아 삭제 후 최신 자료 업로드

### 업로드가 "실패"로 뜸

- 파일 목록의 빨간 글씨(실패 이유) 확인:
  - "스캔된 이미지 PDF" → 글자가 아닌 사진으로 된 PDF는 못 읽음. 원본(글자 있는 버전)을 구하거나 내용을 텍스트로 옮겨서 업로드
  - "50MB 초과" → 영상이면 자막(.vtt)으로, 문서면 나눠서
- 점 3개(⋮) → **재처리** 로 다시 시도 가능

### 챗봇이 처음에만 느림 (몇 초 걸림)

정상입니다. 오래 안 쓰면 시스템이 잠들었다 깨어나는 시간입니다. 연속으로 쓰면 빨라집니다.

---

## 📞 개발 담당자에게 연락해야 하는 경우

- "오류" 항목이 계속 쌓일 때 (시스템 문제)
- 로그인이 안 될 때 (비밀번호 초기화 필요)
- 새 운영자 계정이 필요할 때
- 영상을 자막 없이 통째로 처리해야 할 때 (특수 처리 필요)

## 📚 용어 사전 (참고)

| 용어 | 뜻 |
|---|---|
| **청크(chunk)** | 자료를 검색하기 좋게 자른 작은 조각 (자동으로 만들어짐) |
| **임베딩** | 컴퓨터가 글의 의미를 이해하도록 숫자로 바꾸는 과정 (자동) |
| **RAG** | "자료에서 찾아서 답한다"는 이 챗봇의 작동 방식 이름 |
| **출처** | 답변 끝의 "📚 출처: ○○.pdf" — 어느 자료를 보고 답했는지 |
| **카테고리** | 자료 분류 (오픈마켓 가입 / 사업자등록 / 강의자료 / 도구 가이드 / 기타) |
